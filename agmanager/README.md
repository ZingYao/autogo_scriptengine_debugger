# AG 版本管理功能说明

## 功能概述

AG 版本管理功能提供了对 AutoGo SDK (ag) 的自动下载、更新和版本管理能力。

## 主要功能

### 1. 版本检测
- 自动检测当前系统是否安装了 AG
- 如果未安装，提示用户进行安装
- 显示当前 AG 版本号

### 2. 版本列表查看
- 从远程服务器获取所有可用版本
- 显示每个版本的发布日期和更新内容
- 标记当前正在使用的版本

### 3. 自动更新
- 一键更新到最新版本
- 显示下载进度条
- 自动备份旧版本（以版本号命名）
- 自动安装新版本到指定目录

## 使用方法

### 在 TUI 中使用

1. **查看 AG 管理菜单**
   - 在主菜单中选择 `(8) AG 更新`
   - 或按数字键 `8` 跳转，然后按回车确认

2. **查看版本列表**
   - 在 AG 管理菜单中选择 `(l) 查看版本列表`
   - 显示所有可用版本及其更新内容

3. **更新到最新版本**
   - 在 AG 管理菜单中选择 `(u) 更新到最新版本`
   - 或在版本列表中按 `u` 键
   - 系统会自动下载并安装最新版本

4. **查看当前版本**
   - 在 AG 管理菜单中选择 `(c) 查看当前版本`
   - 显示当前安装的 AG 版本号

## 技术实现

### 文件下载源
- **版本信息**: `https://autogo-1257133387.cos.ap-shanghai.myqcloud.com/changelog.md`
- **SDK 下载**: `http://168.138.164.80:7001/files/AutoGo/sdk/`

### 支持的平台
- `mac_arm` - macOS ARM (M1/M2)
- `mac_amd` - macOS Intel
- `win_x64` - Windows x64
- `linux_x64` - Linux x64

### 文件命名规则
```
{platform}_{version}
例如: mac_arm_1.0.0
```

### 安装目录
默认安装目录: `~/.autogo/`

安装文件结构:
```
~/.autogo/
├── ag              # 当前使用的 AG 可执行文件
├── ag.version      # 版本号文件
├── ag_1.0.0        # 备份的旧版本
└── ag_0.9.0        # 备份的更早版本
```

## 进度条显示

下载过程中会显示实时进度条:

```
████████████████████░░░░░░░░░░░░░░░░░░░░░░░░░░ 50%
```

进度条长度: 50 个字符
更新频率: 实时更新

## 错误处理

1. **网络错误**: 显示友好的错误提示
2. **权限错误**: 提示用户检查目录权限
3. **版本检测失败**: 提示用户手动下载

## 快捷键

在 AG 管理界面:
- `l` - 查看版本列表
- `u` - 更新到最新版本
- `c` - 查看当前版本
- `q` 或 `ESC` - 返回上级菜单

## 注意事项

1. 更新前会自动备份当前版本
2. 下载失败不会影响当前安装的版本
3. 建议在网络良好的环境下进行更新
4. 更新后需要重启项目才能使用新版本
